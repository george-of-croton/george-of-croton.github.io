Scope in Javascript

What Does it Mean?

In javascript, "scope" refers to how much of the code we want a variable to affect. There are two types of scope, "global scope" and "local scope". A variable with global scpope is any variable that is not defined inside a function, and this variable can be referenced or affected by any line of code in the script. A variable with local scope is any variable defined inside a function, this variable can only be used inside that function.
